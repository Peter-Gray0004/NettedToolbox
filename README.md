# Overview
A networking toolbox which gets your public IP, shows each NIC's addresses (MAC, private IPV4 and IVP6) as well as the ability to release/renew IP addresses and flush your DNS.

# How to run
* Open the .sln file

* Add the dependency to the project

* Build and run

# Dependencies
Libcurl
